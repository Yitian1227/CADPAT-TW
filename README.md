本專案以「迷彩」作為視覺基底，將地雷埋藏後形成的坑洞概念轉化為隱藏在圖像裡的深度資訊。
呈現「看似正常、但其實暗藏危機」的視覺語言，在特定的眼睛互動情境下，才會呈現出隱藏的地雷資訊。
